# HC Scaling Solutions Color Palette

Based on the colors extracted from `branding/logo/hc-logo.svg`, here's the suggested color palette for website and branding design.

## Primary Colors

- **Primary Text** (#000080) - Primary brand color, for headings and key text elements
- **Primary Accent** (#000064) - Secondary dark color, for accents and interactive elements

## Secondary Colors

- **Secondary Accent** (#000098) - For subheadings and secondary accents
- **Tertiary Accent** (#00008c) - For secondary buttons and highlights

## Accent Colors

- **Bright Accent** (#00c0fc) - Bright accent color for CTAs and interactive elements
- **Light Accent** (#0000b4) - Lighter accent for links and highlights

## Neutral/Background

- **Light Background** (#ffffff) - Primary light background
- **Subtle Background** (#f3f4f6) - For hover states and subtle backgrounds
- **Light Border** (#e5e7eb) - For borders and dividers
- **Dark Border** (#374151) - For dark theme borders and hover states
- **Dark Background** (#1a1a1a) - Primary dark background

## Usage Guidelines

This palette maintains the professional, tech-focused feel of the logo while providing sufficient contrast and variety for web design. Use the Bright Accent (#00c0fc) sparingly as the primary accent to draw attention to important actions.
